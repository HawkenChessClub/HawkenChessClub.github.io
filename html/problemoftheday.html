Problems due 11/20:

Problem #1:

For this problem, create a segment of code that takes a user input and stores it as a variable r, the radius of the circle and prints out the diameter and circumference of the circle.

Problem #2:

For this problem, ask the user for two inputs. One should be the number of sides for a regular polygon, and one should be the side length. Then print out the area of the polygon using this formula, Area = (s2n)⁄(4*tan(180⁄n)) , in which s = side length, and n = number of sides.

Problem #3:

For this challenge, you should try to create a calculator which can convert from Ballpoint Pens to Rollerball Pens. The conversion ratio is x Ballpoint pens = log20(12xe) Rollerball pens. Have a user input a number of ballpoint pens and get rollerball pens, or input a number of rollerball pens and get ballpoint pens. This conversion should be exact (not a decimal approx.)

Problem #2

        import java.util.Scanner;
import java.lang.Math;
class problem2{
    public static void main(String[] args){
        Scanner scan = new Scanner(System.in);
        System.out.println("Enter the number of sides");
        int n = scan.nextInt();
        System.out.println("Enter a side length");
        int s = scan.nextInt();
        System.out.println("Area: " + Math.round(((Math.pow(s,2)*n)/(4*Math.tan(Math.toRadians(180/n))))*100.0)/100.0);
    }
}
